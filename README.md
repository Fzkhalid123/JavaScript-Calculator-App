# JavaScript Calculator App

A simple and interactive calculator built using **HTML, CSS, and JavaScript**.  
Supports basic arithmetic operations with a clean UI.

---

# Features

- Basic arithmetic: addition, subtraction, multiplication, division  
- Support for decimal numbers  
- Clear button to reset calculations  
- Interactive UI with live display updates on button clicks  

---

# Tech Used

- **HTML** – Structure of the calculator  
- **CSS** – Styling and layout  
- **JavaScript** – Calculator functionality, event handling, and safe calculation logic
